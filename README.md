# monorepo-skeleton

This is a skeleton template for making a monorepo using `lerna` and `yarn`.

It uses the following stuffs:

 * typescript
 * lerna
 * yarn 
 * eslint (standard-with-typescript)


## Note

Make sure to replace `monorepo-skeleton` in the config files:

 * `package.json`
 * `tsconfig.json`
